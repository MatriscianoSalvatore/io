Keiken - Android app

Keiken is an Android app that connects users by letting them create and book "experiences".
Keiken experiences are activities and events created by the users available in the whole world.
Apk at https://keikenofficial.github.io/keiken-web.
Code at http://github.com/keikenofficial/keiken-android.
App made in team with Federico Andrea Tondini and Mattia Bigi for the exam of Software Engineering at University. For the development we used Java, Android Studio, Git, Github, Firebase, Firebase Authentication (Facebook, Google and Email), Firebase Storage, Cloud Firestore database, Crashlytics and Test Lab.

The main component of the app's User Interface is the Backdrop: a new Material Component by Google, but it's still not availale as an Android component, so we implemented our own extending a LinearLayout and a BottomSheetBehavior.

February - October 2019.
